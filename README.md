## What's Inside

This repository contains [default community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file) that GitHub automatically applies to all repositories in the [open-octopus](https://github.com/open-octopus) organization that don't have their own versions.

| File | Purpose |
|------|---------|
| `LICENSE` | MIT License |
| `CODE_OF_CONDUCT.md` | Contributor Covenant v2.1 |
| `CONTRIBUTING.md` | How to contribute to OpenOctopus projects |
| `SECURITY.md` | Security vulnerability reporting policy |
| `ISSUE_TEMPLATE/` | Bug reports, feature requests, realm package requests |
| `PULL_REQUEST_TEMPLATE.md` | Standard PR checklist |

## Organization Profile

The `profile/README.md` powers the organization's landing page at [github.com/open-octopus](https://github.com/open-octopus).
